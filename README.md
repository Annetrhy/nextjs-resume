# Next.js Resume Website

## Description
This project is a personal resume website built using Next.js (App Router) and Tailwind CSS. It recreates my resume as a clean, responsive webpage.

## Technologies Used
- Next.js
- React
- Tailwind CSS

## How to Run Locally
1. Install dependencies:
   npm install

2. Start the development server:
   npm run dev

3. Open in browser:
   http://localhost:3000

## Features
- Fully structured resume layout
- Sections include:
  - Summary
  - Education
  - Experience
  - Skills
  - Achievements & Certifications
  - Leadership & Involvement
- Styled using Tailwind CSS utility classes
- Clean and readable layout optimized for desktop

## Project Structure
- All resume content is located in:
  app/page.tsx
